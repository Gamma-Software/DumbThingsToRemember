## Docker

### DNS

I installed on my server a DNSÂ resolver. The docker engine parameter must be updated to add the ip of the dns server. In the */etc/docker/daemon.json* file
``` json
{
  "dns": ["192.168.1.34", "8.8.8.8"]
}
```

### Store the docker images on another partition

Just override the data-root parameter in */etc/docker/daemon.json*
``` json
{
  "data-root": "/mnt/data/docker_images"
}
```
